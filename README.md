# Annoying Fly

**Annoying Fly** is a lightweight, importable JavaScript module that adds a playful "fly" to any webpage. The fly reacts to user interactions by chasing and circling elements you click on. This project is designed to be simple, fun, and easily extendable for future features.

---

## Features

- Adds a fly that moves across your webpage.
- The fly chases elements you click on.
- Circles elements after reaching them.
- Easy to integrate with just a single import.

---

## Installation

### To use Annoying Fly, simply add the following script to your webpage:

```html
<script type="module">
  import { initFly } from 'https://michaelgalloway404.github.io/AnnoyingFly/fly.js';
  initFly();
</script>
```
### If that give you trouble just use:  
```html
<script type="module"> 
    import { initFly } from 'https://cdn.jsdelivr.net/gh/MichaelGalloway404/AnnoyingFly@main/fly.js'; 
    initFly(); 
</script>  
```
### Or try it out on the is test page: https://michaelgalloway404.github.io/AnnoyingFly/
